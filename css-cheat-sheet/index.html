Semantic tags

Semantic HTML means to use meaningful tags to describe a part or section of the web page, allowing the improvement of SEO, accesibility and code clarity, for example:

Tag Description
<header> Defines the introductory content of a section or page
<nav> Defines a set of navigation links
<main> Contains the central content of a document
<aside> Represents content that is related to the information around it
<footer> Defines a footer of a section or page
<figure> Encapsulates media content, typically with a caption (<figcaption>)
<article> Represents a self-contained piece of content that could be distributed and reused independently
<section> Defines a section in a document

Tables

Tables are used to organize and display data in a structured form by using rows and columns. Let's comprobe the essential tags to make a table:

Tag Description
<table> The tag that creates a table
<tr> Creates a table row where the cells are contained
<td> Table Data, which is the content of the table
<th> Defines the headers of a table, a good way to classify the data
