HTML

Semantic tags

Semantic HTML means to use meaningful tags to describe a part or section of the web page, allowing the improvement of SEO, accesibility and code clarity, for example:

Tag Description
<header> Defines the introductory content of a section or page
<nav> Defines a set of navigation links
<main> Contains the central content of a document
<aside> Represents content that is related to the information around it
<footer> Defines a footer of a section or page
<figure> Encapsulates media content, typically with a caption (<figcaption>)
<article> Represents a self-contained piece of content that could be distributed and reused independently
<section> Defines a section in a document

Specificity

Specificity is used to define the priority of styling of elements, depending on how an element is selected is the amount of specificity that will adquire.

Let's view 5 ways of defining specificity in CSS

  • Elements

                    
                  h1 {
                    color: white;
                  }

  • Class

                    
                  .myClass {
                    padding: 1rem;
                  }

  • ID

                    
                  #myId {
                    display: block;
                  }

  • Inline selectors

                    
                  <p style="color: white;">Hello World<h1>

Box Model

The box model represents how the size of any element is calculated based on 4 properties. In addition, it conveys that any HTML element functions as a box.

  • Margin: Defines de space around an element

  • Border: Defines the line around an element

  • Padding: Defines the space inside an element

  • Content: There's defined the width and height of the content

Display property

The display property specifies the type of display that is used by an element which is shown on the browser once it's added to the document.

In order to use it, it's written: display: block

Let's review the most known of them:

Display Description
Block Occupies the entire width of the container
Inline Fills the container with the enough amount of space
Inline-block It allows to have an inline behaviour but with the allowing of setting the width and height of the element
None The element is not displayed
Flex It uses the flexbox model as a flexible way to display elements
Grid It uses the grid model in order to create complex layouts
