HTML

Semantic tags

Semantic HTML means to use meaningful tags to describe a part or section of the web page, for example:

Tag Description
<header> Defines the header section of the page where there is usually the title, logo, and navigation
<nav> Defines the navigation section
<main> Defines the main content
<aside> Defines an aside
<footer> Defines the footer
<figure> Defines a figure
